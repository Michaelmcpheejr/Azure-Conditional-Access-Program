# Monitoring and Alerting Triggers

## Triggers for Investigation
- Repeated MFA failures above threshold
- Any break-glass usage
- Admin activity outside normal hours
- Any change to Conditional Access policies
- Any change to privileged role assignments or admin RBAC groups

## Expected Responses
- Validate user identity and device context
- Review recent policy and role changes
- Confirm break-glass use had approval and post-use steps completed
- Escalate if any unacceptable failure condition is observed
